# Station Echo

Žánr: 3D platformer, hádanky

Hlavní herní mechanikou je změna vektoru gravitace na stanici pomocí tlačítek a dalších interaktivních objektů.

Postavu lze ovládat pomocí kláves WASD, myši a klávesy F pro interakci.  
Hra by měla implementovat hlavní mechaniky klasických 3D platformerů, jako jsou:
* Pohyblivé plošiny
* Super skok
* Parkour
* Tlačítka a páky, které otevírají dveře
* Pasti
* Animace
